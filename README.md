# Music Playlist Data Analysis

This project analyzes a music playlist dataset using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Project Steps
1. Import dataset
2. Clean missing values
3. Remove duplicates
4. Analyze top artists
5. Analyze popular genres
6. Visualize data
7. Find insights and conclusions

## Key Insights
- The dataset contains music tracks from multiple genres.
- Popular genres include Pop, K-pop, Chill, and Sad.
- Audio features like danceability and energy help understand song popularity.
